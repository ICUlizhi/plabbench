# Kimi Code 使用指南

本指南介绍如何使用 Kimi Code 完成北大计算机课程 Lab。

## 安装与配置

### 安装

访问 [Kimi Code 官网](https://kimi.moonshot.cn/) 下载并安装。

### 配置

1. 启动 Kimi Code
2. 登录 Kimi 账号
3. 配置 IDE 集成（可选）

## 基本使用

### 启动

```bash
# 在项目目录启动
kimi
```

### 常用命令

参考 Kimi Code 内置帮助获取最新命令列表。

## 与 Plabbench 配合使用

使用方式与 Claude Code 类似：

```bash
cd courses/pku-os
kimi
```

然后直接提出你的问题或请求。

## 对比 Claude Code

| 特性 | Claude Code | Kimi Code |
|------|-------------|-----------|
| 上下文长度 | 200K tokens | 200K tokens |
| 代码生成 | 优秀 | 优秀 |
| 中文支持 | 良好 | 优秀 |

## 提示词建议

由于 Kimi 对中文理解较好，可以直接使用中文描述复杂问题。
